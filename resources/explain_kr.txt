[운수 좋은 작명소 간단 설명서]

■ 메인 메뉴
1. 작명하기 - 성별, 이름 타입, 성, 글자수를 선택하여 이름 추천 받기
2. 이력보기 - 이전에 만든 이름 기록 확인하기
3. 설명보기 - 프로그램 사용 방법 안내
4. 환경설정 - 언어 변경 및 개발자 정보 확인
5. 종료하기 - 프로그램 종료

■ 작명 과정
1. 성별 선택: 여자 또는 남자
2. 이름 타입: 귀여운, 강한, 부드러운, 전통적인, 2025년 핫한
3. 성(姓) 입력: 김, 이, 박 등
4. 글자수 선택: 외자, 2글자, 3글자
5. 추천 이름 중 선택 또는 다시 생성

■ 유용한 팁
- 언어는 한국어, 영어, 일본어 중 선택 가능
- 이름 이력은 자동 저장됨
- 추천 이름이 마음에 들지 않으면 다시 생성 가능

좋은 이름을 찾으세요!
